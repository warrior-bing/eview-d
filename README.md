# EUI 组件库

UI components for EApp


## 初始化

#### 1、使用UI组件

```
npm install eview-d
```

使用组件的页面json 文件中添加引用即可，例如：

```
"usingComponents": {
  "eui-view": "eview-d/eview/components/view/view"
}
```

#### 2、预览

项目clone 到本地，使用钉钉/支付宝开发者工具打开即可预览

组件使用说明：components/**/**.js 文件


#### 3、目录结构

```
pages         编写Exemple
components/*  组件
```
